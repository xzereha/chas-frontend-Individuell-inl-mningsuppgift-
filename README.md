# Webstore

This is a small demo site for a webstore created as an assignment for a webcourse in accessability and performance.

The site is a fully static site that does not currently have any functionality and instead focuses just on the frontend.

The code is split into 3 pieces, a static HTML file for the website, one javascript file that simulates actual work being done, and one css file for all of the styling of the webpage.
